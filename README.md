# Spam Email Classifier

This is a simple machine learning project that classifies SMS messages as spam or not spam.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Concepts Used
- TF-IDF (text to numerical features)
- Logistic Regression
- Train/Test Split

## Features
- Predicts whether a message is spam or not
- Accepts user input
- Works on real dataset (spam.tsv)

## How to Run

1. Install required libraries:
pip install pandas scikit-learn

2. Run the program:
python SpamClassifier.py

## Example

Input:
win free money now

Output:
Spam
